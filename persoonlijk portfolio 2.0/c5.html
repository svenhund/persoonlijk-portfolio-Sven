Week 5 : API

In de workshop van deze week stond 'API' (Application Programming Interface) centraal, een API is eigenlijk de motor van een website of applicatie onder de motorkap. De API zorgt er namelijk voor dat een verzoek van een website, zoals bijvoorbeeld informatie over mogelijk vliegtickets, door de website van in dit geval een reisbureau wordt opgevangen. In dit geval zal de website van het reisbureau alle mogelijk opties voor vliegtickets terugsturen naar de website waarop de zoekopdracht ingediend werd. Een API is dus is een software-interface die het mogelijk maakt dat twee applicaties met elkaar kunnen communiceren.

API opdrachten

Om zelf te oefenen met het gebruiken van API kregen wij in deze workshop twee verschillende opdrachten:

OpenWeather : Nijmegen

De eerste opdracht bestond eruit om het actuele weer van één plaats op de wereld weer te geven. Ik koos in dit geval natuurlijk voor Nijmegen en heb deze link dan ook in de code zo gelegd dat Nijmegen weergegeven werd. Dit gaf het volgende resultaat.

COVID-rates : Nederland

Bij de tweede opdracht was het doel om de actuele COVID-19 gegevens van Nederland weer te geven op een webpagina, na dit in HTML goed gecodeerd en geconnect te hebben met de API. Dit gaf het volgende resultaat.

Ervaring

Persoonlijk vond ik dit thema ontzettend interessant en gaf dit enorm veel nieuwe inzichten. Zo zag ik op deze manier in waar veel websites of applicaties hun actuele informatie vandaan halen en hoe zij deze link leggen om dit bij hunzelf weer te kunnen geven. Daarnaast was het ontzettend leuk om zelf te zien dat mij dit ook lukte. Vooraf had ik al nooit verwacht zo ver te komen als bedrijfskundige. Nadat ik erachter kwam dat ik dit zo leuk vond heb ik besloten zelf nog een keer een andere API te integreren op mijn website. Meer hierover hieronder...

Map : Ooij

Als extra uitdaging en vooral ook plezier voor mezelf besloot ik om nog één keer met API's aan de slag te gaan. Hierbij ben ik rond gaan kijken naar welke API ik zou willen toevoegen. Uiteindelijk kwam toen in mij het idee op om maps te integreren en hierbij een marker te zetten op één van mijn twee woonplaatsen, namelijk de Ooij in de Ooijpolder. Na flink wat gestooi en gesteggel is dit mij gelukt en leverde dit het volgende resultaat.

(Een klein minpuntje aan dit resultaat is dat ik geen volwaardig Google account heb en niet mijn rekeningnummer hier zomaar in wilde vullen en er vandaar "For Development Purposes Only" door de map heen staat. Desondanks ben ik alsnog blij met het resultaat en klopt de locatie ook en staat deze precies op Ooij)
