04 : Connection

Voor de leerlijn Connection was het doel om een eigen portfolio in website vorm te maken. Hiervoor hebben wij in de eerste 6 weken een flink aantal workshops gehad. Thema's die hier behandeld werden zijn o.a. HTML, CSS, Javascript, SQL & Non-SQL Databases, API en Machine Learning. Deze aangereikte kennis bood de basis om deze website te kunnen maken. Daarnaast hadden we samen met de klas ook nog een groepsproject waarin wij gezamenlijk een website dienden te vormen. Hiervoor zat ik bij de designers-groep, deze activiteiten zult u hier dan ook beschreven terug gaan vinden.

Leerdoelen

- Past nieuwe technologieën op het gebied van web, data, cloud en AI toe op een praktijkcasus (SC21) - Plaatst de casus in een business context (SC22)
